# crystal-beacon-lab

A small lab repo for **safe beacon telemetry** experiments (agent → local collector) with an emphasis on observability and reproducibility.

## Why it matters
It’s useful to study periodic network/telemetry patterns in a controlled environment to build better detections and understand false positives.
